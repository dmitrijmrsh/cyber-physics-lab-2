# cyber-physics-lab-2

## Инженерная задача: Ollama + Qwen2.5:0.5b

### Параметры запуска инференса
- Сервер: `http://localhost:11434`
- Модель: `qwen2.5:0.5b`
- Количество запросов: `10`
- Время фиксации: `2026-04-23 01:16:23`
- Скрипт: `inference.py`
- Файл с результатами: `report.md`

### Как запустить скрипт
```bash
python -m venv .venv
source .venv/bin/activate        # Linux / macOS
# .venv\Scripts\activate         # Windows

pip install -r requirements.txt

python inference.py
```
